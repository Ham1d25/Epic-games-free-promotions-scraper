# Epic Games Free Promotion Scraper

A simple scraper that collects information about **Epic Games Store free game promotions**.
It helps you track current free games, promotion dates, and store links automatically.

---

## Features

- Detects current free games on Epic Games Store
- Extracts:
  - Game title
  - Promotion start date
  - Promotion end date
  - Store URL
- Lightweight and easy to extend
- Can be automated to run weekly

---

## Requirements

- Python 3.8+
- Libraries:
  - requests
